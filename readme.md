## задача 2
### текст задачи
Разработать класс с тремя целочисленными полями. Создать конструктор копирования. 
Разработать метод, вычисляющий максимальное из полей. Перегрузить метод ToString() для 
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить 
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать 
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и 
другие методы базового и дочернего классов
### Алгоритм решения
threemarks:
1) в классе объявлено строковое поле comment, которое хранит комментарий.
Поле закрытое, поэтому доступ к нему осуществляется через методы класса
2) реализованы конструкторы: без параметров, с параметром и конструктор копирования
3) для работы с полем используется свойство Comment, которое позволяет получать и изменять значение комментария
4) метод AddExclamations() добавляет три восклицательных знака в начало строки, изменяя комментарий
Studentsmark:
1) класс Studentsmark наследуется от ThreeMarks и добавляет три поля _firstMark, _secondMark, _thirdMark, которые хранят оценки
2) реализованы конструкторы: без параметров, с параметрами (для задания оценок и комментария) и конструктор копирования
3) для работы с оценками используются свойства FirstMark, SecondMark и ThirdMark
4) метод IsPassed() проверяет, сдал ли студент (все оценки ≥ 3), а GetMaxMark() находит максимальную оценку

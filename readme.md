### текст задачи
Разработать класс с тремя целочисленными полями. Создать конструктор копирования. 
Разработать метод, вычисляющий максимальное из полей. Перегрузить метод ToString() для 
формирования строки из полей класса. Реализовать дочерний класс (его содержание предложить 
самостоятельно и описать в решении: какой содержательный смысл имеют поля; реализовать 
конструкторы; предложить и реализовать 2-3 метода). Протестировать все конструкторы и 
другие методы базового и дочернего классов
### Алгоритм решения
1) Попадаем в метод Main, выводим  "=== тестирование threeMarks ===" и пустую строку
2) пользователь вводит три оценки через метод ReadMark,
проверяется, что введено число, значение должно быть в диапазоне от 2 до 5,
при ошибке ввод повторяется
3) создаётся объект ThreeMarks, в него передаются оценки и комментарий и выводится на экран
4) создаётся и выводится его копия
5) считается ср. балл (GetAverage)
6) создаётся объект дочернего класса StudentMarks и также выводится его копия
? )IsPassed() - определяет сдал ли студент
GetMaxMark() - находит макс. оценку
HasExcellent() - проверяет наличие 5
8) вызывается метод GetAverage()
9) выводятся все результаты
